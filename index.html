Comment ça marche ?
Ce QR code pointe vers cette page (votre GitHub Pages).
À chaque scan, le compteur s'incrémente et l'utilisateur est
automatiquement redirigé vers serrure-metal-art.fr.
Les données sont conservées localement dans le navigateur.

Pour un suivi multi-appareils, connectez ce script à un backend ou Firebase.
